Nome do Projeto: LOGMOBO

Descrição: Sistema para alertas via e-mail de aparelhos inativos contidos na plataforma Urmobo.

Instruções de Instalação: Necessário realizar a instalação das bibliotecas contidas no arquivo "requirements.txt" ou se preferir utilizar o ambiente virtual através do Conda (pasta VM)

Uso: 

* Se possível revisar o código para incluir as modificações necessárias

* Necessário acesso as configurações do programa para incluir endereço de e-mail para envio das notificações;

* Necessário preenchimento da API do Urmobo para consulta via banco de dados;

Contribuições: Caso de dúvida entrar em contato via Linkedin.
